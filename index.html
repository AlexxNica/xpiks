Xpiks is a free and open source keywording tool for microstock photographers and illustrators with no hidden fees. It's useful if you're uploading your photos to several different stocks and you're tired with copy and pasting keywords from one stock to another. Also you can upload your images/photos to different stocks using Xpiks!

Features:

  • ability to keyword your images as well as set additional XMP/IPTC metadata (title, description, author)
  • can work with separate images as well as directories
  • reading/writing of metadata with ExifTool
  • uploading selected images to list of FTP hosts (created by user) (upload uses Curl inside)
  • ability to upload vector eps files with images with same name (useful for illustrators)
  • works under Linux, OS X and Windows
  • written in C++ with Qt 5.4
  • source code released under GPL v3 license
  • prebuilt packages for OS X Yosemite and Windows 8 x64
