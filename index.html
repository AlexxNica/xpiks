Xpiks is a free and open source keywording and uploading tool for microstock photographers and illustrators. It was written from scratch with aim to make it more usefull, convenient and quicker than existing tools.

Main features:

  • keywording capabilities as well as setting additional XMP/IPTC metadata (title, description, author)
  • uploading images to Stock Agencies' FTP hosts (keeping encrypted list of credentials)
  • keywords suggestion based on Shutterstock API or on the local library
  • search through loaded images by any metadata (keywords, description, title)
  • ability to upload vector (eps/ai) files with images with same name (useful for illustrators)
  • automatically zip vectors and previews if needed
  • checks for potential problems before upload (insufficient resolution, missing metadata etc.)
  • unsaved work is backed up so you can continue work later without actual writing metadata to images
  • ExifTool and Curl binaries used as dependencies
  • written in C++ with Qt 5.4
  • source code released under GPL v3 license
  • prebuilt packages for Debian amd64, OS X Yosemite and Windows 8+ x64
