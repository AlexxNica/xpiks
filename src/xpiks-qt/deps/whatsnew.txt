This is Xpiks 1.3 release. 

It provides following features:

- autocompletion for keywords
- localization to 6 languages (Ukrainian, Portugese, Spanish, Catalan, Russian, French, German)
- keywords suggestion from iStock and Fotolia
- brand new color theme
- dozen of predefined FTP addresses of stocks
- basic hotkeys support
- watching for removal of images and vectors on disk
- improvements for keywords suggestion algorithm
- fixes in the UI (unification, effects for popups)

And bugfixes:

- scrolling jumps
- incorrect warnings for non-existent properties
- preliminary fix for paths with unicode characters in Windows


Enjoy!
